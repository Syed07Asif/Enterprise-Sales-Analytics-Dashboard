# 📊 Enterprise Sales Analytics Dashboard

## Overview

Enterprise Sales Analytics Dashboard is an end-to-end Business Intelligence project built using **Power BI, PostgreSQL, SQL, and DAX**.

The project simulates a real-world enterprise sales environment and provides actionable insights into revenue, profitability, customer behavior, product performance, regional sales, and promotional effectiveness.
